# thumbs
Real-time feedback and analytics for lecturers to iterate on their content.

### Tech Stack
1. Node / Express
1. React / Redux
1. HTML / Sass
1. Victory (D3)
1. PostgreSQL / Sequelize
1. Socket.io
1. Docker
1. CircleCI
1. AWS EBS/RDS/S3

## Quick Start


### Fork/Clone
```
git clone https://github.com/3rn/thumbs.git
```

### Install Dependencies
```
yarn install
```

### Run
```
yarn start
```
